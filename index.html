About

Layanan yg kami tawarkan adalah sebuah layanan yg berbasis Desktop sekaligus berbasis Web & Mobile, ketiga layanan ini dapat mengakses database yg sama, contoh: nama database yg diakses bersama dari ke3 layanan tsb yaitu db_bersama, baik aplikasi desktop web maupun mobile sama2 bisa mengakses db_bersama tsb. Server database dari layanan yg kami tawarkan ini bisa menggunakan server milik pribadi tanpa harus menggunakan layanan hosting pihak ke3.

Fungsi layanan yg kami tawarkan ini mirip dengan fungsi API (Application Program Interface), fungsi API ini memungkinkan aplikasi untuk saling berbagi data dan layanan, sehingga memungkinkan aplikasi untuk terintegrasi dan berinteraksi satu sama lain. Walaupun layanan ini mirip dengan fungsi API, tetapi layanan kami ini menggunakan/mengakses database/server secara langsung menggunakan akses VPN.

Dengan menggunakan server milik pribadi, maka keamanan database akan terjaga. Server pribadi ini bisa menggunakan "Semi Server" atau PC/Laptop biasa yg hanya aktif kurang dari 24jam/perhari atau bisa menggunakan "Full Server" yg bisa aktif 24jam.

VPN

VPN adalah Jaringan Pribadi Virtual, yang berarti jaringan ini dimulai dari sisi pengguna. Virtual Private Network menjadi salah satu solusi yang paling ampuh untuk melindungi privasi dan keamanan internet. Selain dalam hal privasi dan keamanan, VPN juga memungkinkan kita mengakses lebih banyak konten di internet. Kita bisa mengubah lokasi virtual dengan VPN dan membuka website yang biasanya hanya bisa diakses di wilayah tertentu saja, umumnya di luar negeri.

VPN diaktifkan di perangkat yg menggunakannya, semua request untuk mengakses website akan diteruskan melalui tunnel terenkripsi dan dirutekan melalui server VPN sebelum mencapai server target.

VPN di instalasi di semua perangkat yg terhubung dengannya. Semua perangkat yg terhubung dengan VPN ini terkoneksi dengan internet, atau dengan kata lain VPN adalah jaringan internet pribadi (terenkripsi) didalam jaringan internet global.

Kami menggunakan layanan VPN ini dalam mengakses database aplikasi yg kami kerjakan, dengan menggunakan tunnel VPN ini kami menjamin kemananan database anda.

Desktop Web & Mobile

Aplikasi DWM (Desktop Web & Mobile) merupakan 3 aplikasi yg berbeda Platform.

Aplikasi berbasis Desktop merupakan aplikasi yang dijalankan pada masing-masing komputer atau klien. Aplikasi berbasis desktop harus diinstall terlebih dahulu ke dalam komputer agar dapat digunakan. Aplikasi jenis ini mampu beroperasi tanpa terhubung dengan koneksi internet (offline), tetapi bisa mengakses server databasenya melalui jaringan internet (online) menggunakan VPN. Aplikasi berbasis Desktop hanya dapat diakses melalui perangkat PC/Laptop.

Aplikasi berbasis Web adalah aplikasi yg berjalan pada Web Browser & memerlukan jaringan internet untuk menjalankannya. Aplikasi berbasis web based dapat diakses dari berbagai perangkat (PC/Laptop, Smartphone, Tablet) dengan hanya menggunakan web browser.

Aplikasi mobile atau disebut juga dengan mobile apps adalah aplikasi yang dibuat serta dikembangkan dengan berbagai program yang kemudian disesuaikan dengan perangkat mobile. Pada umumnya aplikasi mobile ini dipakai pada smartphone & juga tablet. Aplikasi mobile ini tidak dikhususkan untuk dioperasikan pada PC atau Laptop, walaupun demikian aplikasi mobile ini bisa juga berjalan pada PC atau laptop menggunakan aplikasi bantu yg dikenal dengan emulator.
